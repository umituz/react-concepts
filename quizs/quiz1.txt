About React

1Q: Why do we need to `import React from “react”` in our files?
1A: React is what defines JSX


2Q: If I were to console.log(page) in index.js, what should show up?
2A: A javascript object. React elements that describe what React should eventually add to the real DOM for us

3Q: What is wrong with this code

const page = (
<h1>Hello World</h1>
<p>Learning React</p>
)

3A: We need JSX to be nested under a single parent element

4Q: What does it mean for something to be “declarative” instead of “imperative”?
4A:
- Declaratives mean I can tell the computer WHAT to do and expect it to handle the details
- Imperatives mean it HOW to do each step.

5Q: What does it mean for something to be ”composable”?
5A: We have small pieces that we can put together to make something larger/greater than the individual pieces
